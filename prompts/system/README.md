# System Prompt Assets

This directory contains high-authority system prompt files used to constrain lesson-generation models.

These prompts should be used together with the project root directive and the constitutional learning documents.

Primary file:
- `constitution-bound-lesson-architect.system.md`

Use system prompts for:
- defining role,
- establishing non-negotiable behavior,
- preserving constitutional fidelity,
- preventing drift into fluff or fragmentation.
